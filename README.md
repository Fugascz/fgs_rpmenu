# Requirements:
* ESX
* [esx_billing](https://github.com/esx-framework/esx_billing)
* [esx_society](https://github.com/esx-framework/esx_society)
* [sqz_carmenu](https://github.com/czsquizer/sqz_carmenu)

# Installation
  1. Put this script to your resources
  2. Add `ensure fgs_rpmenu` to your server.cfg
